## 第一部分：快速上手

### 1. 账户注册

首次使用需要注册一个账户。

1.  在浏览器中打开 **时序同笺** 网站 [`https://sxtj.hxcn.space`](https://sxtj.hxcn.space)。
2.  点击登录页面的“**立即注册**”链接。
3.  填写您的学号、姓名、班级、密码等信息。请确保信息准确，特别是学号，它将作为您的主要登录凭证。
4.  点击“**注册**”按钮完成操作。

![image-20251012022501042](https://gastigado.cnies.org/d/halo20251012csguide/image-20251012022501042.png)

### 2. 登录系统

注册成功后，系统会自动跳转到登录页面。

1.  输入您的**学号**和**密码**。
2.  点击“**登录**”按钮，即可进入您的个人仪表盘。

![image-20251012022546407](https://gastigado.cnies.org/d/halo20251012csguide/image-20251012022546407.png)

## 第二部分：个人课表管理

登录后，您会看到“我的课表”页面。这里是您进行所有个人日程管理的核心区域。

![image-20251012022646835](https://gastigado.cnies.org/d/halo20251012csguide/image-20251012022646835.png)

### 1. 创建你的第一张课表

您可以为不同学期或不同用途创建多张课表。

1.  在“我的课表”页面，点击课表名称下拉菜单。
2.  选择“**新建课表**”。
3.  在弹出的窗口中，输入**课表名称**（例如“大三上学期”），选择**学期开始日期**和**总周数**。
4.  点击“**确认**”，一张新的空白课表就创建好了。

![image-20251012022910929](https://gastigado.cnies.org/d/halo20251012csguide/image-20251012022910929.png)

### 2. 导入课表（核心功能）

手动添加课程很麻烦？别担心，我们提供强大的导入功能。

在课表页面右上角，找到“**更多**”按钮，点击它会展开一个功能菜单，导入、导出、调休等高级功能都在这里。

![image-20251012023530826](https://gastigado.cnies.org/d/halo20251012csguide/image-20251012023530826.png)

#### 2.1 从教务系统一键导入

这是最推荐的方式，可以完美同步您的所有课程。

1.  点击“**更多**” -> “**导入课表**”。
2.  在弹出的窗口中，选择“**从教务系统导入**”。
3.  输入您的**教务系统学号、密码**和**验证码**。
4.  选择是将课程导入到**当前课表**还是**新建一张课表**。
5.  点击“**开始导入**”，稍等片刻，您的所有课程就会自动出现在日历上！

![image-20251012024050880](https://gastigado.cnies.org/d/halo20251012csguide/image-20251012024050880.png)

#### 2.2 从 ICS 文件导入

如果您在其他日历应用（如 Google Calendar、Apple Calendar）中已经有课表，可以导出为 `.ics` 文件再导入。

1.  点击“**更多**” -> “**导入课表**”。
2.  在弹出的窗口中，选择“**从ICS文件导入**”。
3.  点击上传区域，选择您电脑上的 `.ics` 文件。
4.  系统会自动解析文件并导入事件。

![image-20251012024159675](https://gastigado.cnies.org/d/halo20251012csguide/image-20251012024159675.png)

### 3. 手动管理日程

除了导入，您也可以随时手动添加、修改或删除课程、会议、活动等。

#### 3.1 添加日程事件

1.  点击“**更多**” -> “**添加日程**”，或者直接**点击日历上的空白位置**。
2.  在弹出的窗口中，填写事件的**名称、地点、时间、周数、星期**等详细信息。
3.  点击“**创建**”即可。

![image-20251012124759515](https://gastigado.cnies.org/d/halo20251012csguide/image-20251012124759515.png)

#### 3.2 编辑或删除日程事件

1.  在日历上**点击**您想要修改或删除的日程事件。
2.  在弹出的详情窗口中，您可以：
    *   修改事件信息。
    *   点击“**删除**”按钮，将该事件从课表中移除。

![image-20251012125119798](https://gastigado.cnies.org/d/halo20251012csguide/image-20251012125119798.png)

### 4. 导出与分享课表

想把课表分享给朋友，或者在其他设备上查看？

1.  点击“**更多**” -> “**导出课表**”。
2.  您会看到两个选项：“导出到图片(PNG)”和“导出到ICS”。

![image-20251012125518950](https://gastigado.cnies.org/d/halo20251012csguide/image-20251012125518950.png)

#### 4.1 导出为图片 (PNG)

-   选择“**导出到图片(PNG)**”。
-   系统会自动生成一张当前视图的课表截图，您可以直接保存或分享。非常适合打印或发给没有使用本系统的同学。

![2025年秋季（大三下）-课表-2025-10-12](https://gastigado.cnies.org/d/halo20251012csguide/2025%E5%B9%B4%E7%A7%8B%E5%AD%A3%EF%BC%88%E5%A4%A7%E4%B8%89%E4%B8%8B%EF%BC%89-%E8%AF%BE%E8%A1%A8-2025-10-12.png)

#### 4.2 导出为日历文件 (ICS)

-   选择“**导出到ICS**”。
-   系统会生成一个 `.ics` 文件。您可以将此文件导入到任何支持标准日历格式的应用中，如手机自带的日历、Google Calendar 等，实现跨平台同步。

### 5. 管理多张课表

您可以同时拥有多张课表，例如不同学期的课程、个人活动安排等。

-   点击页面顶部的**课表名称下拉菜单**。
-   在菜单中，您可以：
    *   **切换**当前显示的课表。
    *   **新建**课表。
    *   通过“**课表设置**”来**重命名**或**删除**当前课表。
    *   通过“**课表设置**”可以**隐藏**或者**结束**课表，设置隐藏或者结束的课表**团队视图不可见**。

![image-20251012130524133](https://gastigado.cnies.org/d/halo20251012csguide/image-20251012130524133.png)

## 第三部分：高级调休功能

遇到法定节假日或者学校临时调课？强大的调休功能可以帮您轻松搞定。

点击“**更多**” -> “**放假调休**”即可进入调休设置面板。

![image-20251012131229601](https://gastigado.cnies.org/d/halo20251012csguide/image-20251012131229601.png)

### 1. 课程对调（换课）

如果周六需要补周四的课，这个功能就派上用场了。

1.  选择“**对调工作日**”模式。
2.  **源日期**：选择**原本有课**的日期（例如：周四）。
3.  **目标日期**：选择**课程需要调往**的日期（例如：周六）。
4.  点击“**确认对调**”，系统会自动将源日期的所有课程“搬”到目标日期。

![image-20251012132820785](https://gastigado.cnies.org/d/halo20251012csguide/image-20251012132820785.png)

### 2. 设置节假日（放假）

如果某一天需要放假，当天的所有课程都会被自动隐藏。在设置放假操作之前，记得**先进行调休（课程对调）操作**哦。

1.  选择“**设置假期**”模式。
2.  在日历上点击您希望设为假期的**日期**。
3.  点击“**确认放假**”。
4.  您会看到课表上对应日期的课程全部消失了。所有调休记录都会显示在右侧列表中，您可以随时**撤销**。

![image-20251012132136168](https://gastigado.cnies.org/d/halo20251012csguide/image-20251012132136168.png)

## 第四部分：团队协作系统

无论是学生社团、项目小组还是班级，团队功能都能让成员间的日程安排变得透明高效。

点击导航栏顶部的“**团队**”菜单，进入“我的团队”页面。

![image-20251012133057677](https://gastigado.cnies.org/d/halo20251012csguide/image-20251012133057677.png)

### 1. 加入团队

1.  从团队创建者那里获取 **8位邀请码**。
2.  在“**加入团队**”区域，输入邀请码。
3.  点击“**加入团队**”，即可成功加入。

![image-20251012133254742](https://gastigado.cnies.org/d/halo20251012csguide/image-20251012133254742.png)

### 2. 查看团队课表

加入团队后，在“我的团队”页面，您会看到一个团队列表。

![image-20251012133809452](https://gastigado.cnies.org/d/halo20251012csguide/image-20251012133809452.png)

#### 2.1 聚合视图

1.  点击团队卡片上的“**查看课表**”按钮。
2.  您将进入一个全新的日历视图，这里**聚合了所有团队成员的日程**。
3.  不同成员的课程会用**不同的颜色**标记，一目了然。

![image-20251012133835711](https://gastigado.cnies.org/d/halo20251012csguide/image-20251012133835711.png)

#### 2.2 高级筛选

当团队成员很多时，如何快速找到有效信息？

-   在团队课表页面的侧边栏，您可以使用强大的**筛选功能**：
    -   **按成员筛选**：只看特定几个人的课表。
    -   **按班级/年级筛选**：快速找到同班或同年级的成员。
    -   **关键词搜索**：直接搜索成员姓名或课程名称。

![image-20251012133918851](https://gastigado.cnies.org/d/halo20251012csguide/image-20251012133918851.png)

#### 2.3 识别课程冲突

-   在聚合视图中，如果某个时间段有**多个成员都有课**，您可以直接点击该时间段。
-   系统会弹出一个窗口，清晰地列出**所有冲突的课程**、**上课成员**和**地点**，便于您安排团队活动，避开大家都有课的时间。

`[截图：点击一个时间冲突格子后，弹出的冲突详情窗口]`

### 3. 创建团队

1.  在“**创建新团队**”区域，输入您的**团队名称**。
2.  点击“**创建团队**”按钮。
3.  创建成功后，系统会为您生成一个**8位的团队邀请码**。将这个邀请码分享给您的朋友，他们就能加入团队了！

![image-20251012133209353](https://gastigado.cnies.org/d/halo20251012csguide/image-20251012133209353.png)

### 4. 管理你的团队（创建者专属）

如果您是团队的创建者，您将拥有更多管理权限。

1.  在“我的团队”页面，点击您创建的团队卡片上的“**管理团队**”按钮。
2.  进入团队的高级管理面板。

![image-20251012134357724](https://gastigado.cnies.org/d/halo20251012csguide/image-20251012134357724.png)

#### 4.1 添加与移除成员

-   **添加成员**：直接输入成员的**学号**即可精确邀请。
-   **移除成员**：在成员列表中，点击对应成员旁的“**移除**”按钮。

![image-20251012134444533](https://gastigado.cnies.org/d/halo20251012csguide/image-20251012134444533.png)

#### 4.2 转让团队所有权

-   如果您希望将团队交给其他人管理，可以点击**“管理团队”**->“**转让**”。
-   从现有成员中选择一位新的创建者并确认。之后，您将变为普通成员。

![image-20251012134627906](https://gastigado.cnies.org/d/halo20251012csguide/image-20251012134627906.png)

#### 4.3 解散团队

-   如果团队不再需要，可以点击“**解散团队**”。
-   为了防止误操作，系统会要求您**输入团队名称进行二次确认**。此操作不可撤销，请谨慎使用。

![image-20251012134646133](https://gastigado.cnies.org/d/halo20251012csguide/image-20251012134646133.png)

### 5. 退出团队

作为普通成员，您可以随时退出团队。

1.  在“我的团队”页面，找到您想退出的团队。
2.  点击卡片上的“**退出团队**”按钮，并在弹窗中确认即可。

## 第五部分：个人资料设置

点击页面右上角的**您的头像或姓名**，可以进入个人资料页面。

![image-20251012134900648](https://gastigado.cnies.org/d/halo20251012csguide/image-20251012134900648.png)

### 1. 修改个人信息

-   在“个人资料”页面，您可以修改您的**姓名、班级**等信息。
-   修改后点击“**保存更改**”。

### 2. 更换头像

-   点击当前的头像图片。
-   您可以选择**上传新图片**作为头像。

### 3. 修改密码

-   切换到“**修改密码**”选项卡。
-   输入您的**旧密码**和**新密码**，然后确认。
