时序同笺是一款专为山东师范大学师生打造的现代化、全功能课表与日程管理Web应用。无论您是需要管理个人繁多的课程，还是需要与团队成员高效协作，时序同笺都能满足您的需求：

- **个人日程管理**: 多课表管理、一键导入课表（正方教务、ICS文件）、手动创建日程事件、多视图模式（周/月）、导出ICS文件、一键生成课表完整截图、智能调休系统（节假日设置、调休）
- **团队协作系统**: 智能邀请机制、8位随机邀请码、灵活团队管理、团队所有权转让、分层权限控制、三级权限体系、团队课表聚合、时间冲突、高级筛选功能、便捷成员操作、团队生命周期管理
- **管理员功能**: 用户管理、课表管理、团队监控、系统设置、头像上传、存储配置

为了让您更快地熟悉和使用时序同笺，我们为您准备了详细的指引：

如果**您是新用户**，请阅读[《保姆级用户教程》](https://hs.cnies.org/archives/chronosync-nanny-user-tutorial)，一步步了解如何创建和管理个人课表，以及如何加入团队进行协作。

如果**如果您是社团/实验室/组织管理者或活动策划者**，希望高效收集团队成员课表，便于成员协作和时间安排，请阅读[《团队协作教程》](https://hs.cnies.org/archives/chronosync#🎯-团队协作系统)。

如果**您想全面了解时序同笺的每一个功能细节**，请阅读[《用户文档》](https://hs.cnies.org/archives/chronosync#📖-使用指南)。

如果**您是开发者**，并希望自行部署本项目，请阅读[《自托管教程》](https://hs.cnies.org/archives/chronosync#🚀-快速开始)。
